Analysis of EEG & music

Capstone research project to examine if music can be suggested using artificial intelligence based on an individual's state of mind. The project utilized electroencephalography(EEG) data, which measures voltage activity in an individual's brain. The experiment paired two genres of music ( pop & classical) with three different activities. A set of 10 samples were generated for each of the nine experiments ( A sample means a session of conducting an experiment while wearing the EEG headset). Each instance produced a CSV File of 8 by n data points. Each column represented one of the eight sensors on the EEG headset. Python and libraries, including numpy, scikitlearn, and signal were used to conduct various tasks, including data pre-processing, visualization, and machine learning. The project results were inconclusive but serve as a good starting point for future research. Click the github link above to see a repository of the code used in the project.

 

Eavesdrop Content Generator

University of Alberta HackED 2020 Project

Eavesdrop listens to conversations and takes that input and combines it based on nouns and verbs with input from pre-written material eg. a movie script. This modified material is then fed into the gpt-2 algorithm to generate a story. The project was built with python, and uses google cloud tools to run the gpt-2 model.

 

Workplace Scheduling App

MacEwan University Software Engineering Project

This workplace scheduling app was developed for a fictitious restaurant for the MacEwan University CMPT 395 software engineering group project. The application was developed using Android Studio, Java, SQLlite, and GitHub.

Features include:

  • Viewing and creating schedules
  • Setting availability and whether they have been trained to open, close, or both
  • Exporting a week's schedule into a PDF document.
